# Quick Start Checklist

## Setup (5 minutes)

- [ ] Node.js 18+ installed
- [ ] Run `npm install`
- [ ] Create `.env` from `.env.example`
- [ ] Add `OPENAI_API_KEY` to `.env`
- [ ] Run `npm run dev`
- [ ] Open http://localhost:3000

## Deploy (10 minutes)

- [ ] Push to GitHub
- [ ] Sign up to Vercel
- [ ] Import project
- [ ] Add environment variables
- [ ] Deploy

Total time: ~15 minutes
