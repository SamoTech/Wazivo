## ğŸ› Reporting Issues

### Bug Reports

Found a bug? Help us fix it!

**â†’ [Report a Bug](https://github.com/SamoTech/Wazivo/issues/new?labels=bug&template=bug_report.md)**

**Include:**
- Clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)
- Your environment:
  - Browser & version
  - Operating system
  - Node.js version (for local development)

**Example:**

```markdown
**Bug Description:**
CV upload fails with PDF files larger than 2MB

**Steps to Reproduce:**
1. Go to homepage
2. Click "Upload CV"
3. Select a PDF file > 2MB
4. Error appears

**Expected:** File should upload successfully
**Actual:** Error "File too large"

**Environment:**
- Browser: Chrome 120
- OS: Windows 11
```
